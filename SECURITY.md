# Security

This document describes the supply chain security controls in place for this GitHub Action and how consumers can verify its integrity.

## Verifying Attestation

Every release includes a build provenance attestation generated by GitHub's artifact attestation infrastructure. To verify:

```bash
# Download the release artifact and verify its provenance
gh attestation verify dist/index.js --repo <owner>/<repo>
```

This confirms the artifact was built from the claimed source commit in GitHub Actions CI, with no manual tampering.

## Pinning to a Verified Release

For maximum supply chain security, pin this action to a full commit SHA rather than a mutable tag:

```yaml
# Recommended — immutable reference
uses: octo-observability/submit-sbom-action@<full-40-char-commit-sha>

# Acceptable — tag reference (mutable, but attested)
uses: octo-observability/submit-sbom-action@v1.0.0
```

Find commit SHAs for releases on the [Releases page](../../releases).

## Dependency Posture

| Metric | Value |
|--------|-------|
| Direct production dependencies | 3 |

| All maintained by | GitHub |
| Distribution method | Bundled via `@vercel/ncc` — single `dist/index.js` |
| `node_modules` shipped | No |
| Install scripts | Disabled (`.npmrc: ignore-scripts=true`) |

## Supply Chain Controls

| Control | Implementation |
|---------|---------------|
| Exact version pinning | All dependencies use exact versions (no `^` or `~`) |
| Install script prevention | `.npmrc` with `ignore-scripts=true` |
| Stability delay | Renovate waits 3 days before proposing new versions |
| Dependency review | PRs adding vulnerable dependencies are blocked by CI |
| Workflow security linting | zizmor scans all workflow files for security issues |
| Build attestation | Every release attested with `actions/attest-build-provenance` |
| Action ref pinning | All CI/release workflow action refs use full commit SHAs |
| Bundle size limit | Release fails if `dist/index.js` exceeds 5MB |
| Secret scanning | Release fails if common secret patterns are detected in dist |
